## 🔮 How Pig Latin Works (Quick Refresher)

- If a word starts with a vowel → add `yay` to the end  
  `example` → `exampleyay`

- If it starts with a consonant or cluster → move it to the end and add `ay`  
  `pig` → `igpay`  
  `smile` → `ilesmay`
